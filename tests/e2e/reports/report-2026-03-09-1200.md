## Рекомендации

1. **Срочно исправить** страницу настроек — скорее всего нужна проверка на `undefined`/`null` перед вызовом `.toFixed()`, либо API не возвращает ожидаемые данные
2. **Срочно добавить** проверку на самоудаление администратора — бэкенд должен запрещать удаление текущего пользователя
3. **Добавить ErrorBoundary** на уровне роутов для корректной обработки ошибок рендеринга
4. **Автозакрытие модальных окон** после успешных CRUD-операций
5. **Повторно протестировать** загрузку файлов (docs-002, 003, 004, 100) вручную
6. **Обновить тест-кейсы** — привести тексты ожидаемых результатов в соответствие с русской локализацией UI
